1. git clone : adalah cara menyalin projek repository

2. git init: adalah mengecek projek

3. git status : adalah projek sudah terbaca atau tidak

4. git add . : adalah printah menyelek file dari folder

5. git commit -m "isi pesan" : adalah memberikan nama file yang push

6. git push : adalah ekspor projek ke internet

7. git pull : adalah jika projek mengulang ekspor ke internet

8. clear : adalah hapus untuk mengunakan git bash

9. cls : adalah hapus untuk comment promt atau cmd
